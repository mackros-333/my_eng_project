# Инженерный проект: Песочница новичка

## О проекте
Этот проект создан, чтобы освоить не просто написание кода, а **инженерную дисциплину**.

Здесь есть:
- Чистая архитектура (папки `src/` и `tests/`)
- Изолированное окружение (`venv`)
- Защита от ошибок (`try/except`)
- Доказательства работоспособности (`pytest`)
- Красивый код (`black`)
